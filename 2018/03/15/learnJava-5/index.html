内部类

在类文件中再定义一个类，这个类就称为 内部类。

特点：

  1. 含有内部类的类编译会生成两个文件 类名.class 和 类名$内部类.class；
  2. 要生成内部类对象必须首先有一个外部类对象存在，例 A.B b = new A().new B(); 其中B为内部类；
  3. 内部类可以直接使用外部类成员；

匿名内部类 就是没有名字的内部类，例子如下

//匿名内部类
b.fun(new Aa() {
    @Override
    public void doSomething() {
        System.out.println("匿名内部类");
    }
});

其中 Aa 为一个接口，b.fun() 的参数为一个Aa的实现类，此时直接 new Aa() 并实现方法产生的对象就是一个匿名内部类的对象。

多线程和多进程

多进程 ：在操作系统中能（同时）运行多个任务（程序）。（一个进程代表一个程序在运行）

多线程 ：在同一应用程序中有多个顺序流（同时）执行。（线程是进程当中的一个程序执行流程）

线程的执行过程

步骤

  1. 生成线程（new一个线程）；
  2. 进入就绪状态（Runable）；
  3. 获得 cpu 执行后进入运行状态（Running）；
  4. 失去 cpu 后再次进入就绪状态；
  5. 遇到阻塞事件时进入阻塞状态（Blocked）；
  6. 解除阻塞后再次进入就绪状态；
  7. 运行完成后进入死亡状态（Dead）；

创建线程的方法

  • 方式1：定义一个线程类，它继承类 Thread 并重写其中的方法 run(),方法 run() 又称为线程体，在创建对象后使用 start() 进入就绪状态；（由于 Java 只支持单继承，用这种方法定义的类不能再继承其他类）
  • 方式2：提供一个实现接口 Runnable 的类作为线程的目标对象，在初始化一个 Thread 类或者其子类的线程对象时，把目标对象传递给这个线程实例，由该目标对象提供线程体；（常用匿名内部类的方法）

注意：千万不能直接使用 run() 来启动线程，而应该用 start()；在实际开发过程中更倾向于使用接口而不是继承。

线程的简单控制方法

  • 中断线程：
    • Thread.sleep(参数) 线程体当中调用时，当前线程根据参数休眠（阻塞状态）后再进入就绪状态，单位为 ms；
    • Thread.yield() 线程体当中调用时，当前线程在运行状体会让出 cpu 进入就绪状态；
  • 设置程序优先级：(线程的优先级最大是10，最小是1)
    • getPriority() 通过线程对象调用可返回线程的优先级数值；
    • setPriority(参数) 通过线程对象调用，并根据参数设置优先级（可使用 Thread 静态变量）；

注意：设置优先级只能提高运行概率，不能决定是否先运行。

多线程数据安全

在多线程访问同一数据时容易出现数据安全问题，需要使用 同步代码块 synchronized(this/类实例对象) {...} （其中的 this/对象 被称为 同步锁）

注意：每一个线程都有名字，可以通过 setName() 方法设置，通过 getName() 方法获取；可以通过 currentThread() 获取当前正在执行的线程对象的引用；

同步语法

  • 一旦一个线程获得了一个对象的同步锁，这个对象的其他使用该对象同步锁的代码不能被其他线程使用；
  • 将 synchronized 放在方法定义中后该方法就是同步方法；（同步方法的同步锁是 this）
  • 同步锁锁住的是对象；

数组

以一个整型数组为例：

//数组的静态定义方法
int[] arr = {1, 2, 3, 4};

中括号可以放在变量名前，也可以放在变量名后；注意该数组的类型是整型数组类型，它的元素是整数类型；

arr[3] = 10;

通过数组元素下标获取数组的元素（元素下标从0开始）来对数组进行一般操作；

arr.length

通过 length 属性可以获得数组的长度；

//数组的动态定义方法
int[] arr = new int[10];

用 new 来动态创建数组，并在中括号中声明数组的长度大小（这样创建的整型数组其默认值为0）；

//二维数组的定义方法
int[][] arr = {{1, 2, 3}, {4, 5, 6}, {7, 8, 9}};
arr[0][0] = 0;

二维数组中外层数组的元素是内层数组，因此左边的中括号中的下标代表第几个内层数组，右边中括号中的下标代表第几个内层数组的元素；

int[][] arr = new int[3][5];

动态声明二维数组，该声明定义有三个内层数组，每个数组有5个元素；

for (int i = 0; i < arr.length; i++){
        for (int j = 0; j < arr[i].length; j++){
            System.out.println(arr[i][j]);
        }
    }

注意：数组一旦被声明后其长度固定不可改变；数组的元素必须保持一致。
