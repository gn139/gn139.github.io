面向对象

  • 面向对象是一种编程方法；
  • 面向对象是一种思维方式；
  • 面向对象不是一种编程语言；

面向对象思维方式

  • 首先确定谁来做，其次确定怎么做；
  • 首先考虑整体，其次考虑局部；
  • 首先考虑抽象，其次考虑具体；

定义类的方法

class 类名{
    属性；
    方法；
}
  • 属性也叫 成员变量 ，主要用于描述类的状态；
  • 方法也叫 成员方法 ，主要用于描述类的行为；

生成对象的方法

格式：类名 对象名 = new 类ming();

例子：Dog dog = new Dog();

类和对象的关系

“某一只”和“某一类”的区别。对象是类的具体存在，而类是对象的抽象化概念。

对象的使用方法

使用对象调用变量和函数：

  1. 对象.变量
  2. 对象.函数名()

生成多个对象

Dog d1 = new Dog();

匿名对象的使用

可以不定义对象的引用名称，而直接调用这个对象的方法。这样的对象就称为 匿名对象。

例： new Dog().jump();

函数的重载

重载关系判断条件：

  1. 两个函数在同一类当中；
  2. 两个函数的函数名相同；
  3. 两个函数的参数列表不同；

根据参数的不同，来区分使用重载函数。

构造函数

特点：

  1. 没有返回值类型的定义；
  2. 函数名与类名相同；
  3. 只能用 new 来调用构造函数；
  4. 当没有自定义构造函数时，编译器会自动添加默认的构造函数；

作用：

  1. 生成类的对象；
  2. 自定义构造函数，为成员变量赋初始值；

注意：自定义构造函数后编译器不会再添加默认的构造函数。

使用 this 调用成员变量和成员函数

public class Person {

    String name;

    void talk(){
        System.out.println("my name is "+ this.name);
    }
}

this 在代码中代表调用了 talk() 函数的对象。因此，this 多用于区分成员变量和参数，如下：

public class Person {

    String name;

    void talk(String name){
        System.out.println("my name is "+ this.name);
    }
}

调用构造函数

Person(String name){
        this.name = name;
    }

Person(String name, int age){
    this(name);
    this.age = age;
}

this 后跟括号代表调用构造函数的重载函数，具体调用哪个由参数决定。
注意：此处 this 必须是第一条语句。

静态成员变量的语法特点

在声明成员变量时，在类型定义前加上 static 关键字后，此成员变量即为静态成员变量，例：static int i;。

语法特点：

  1. 可以直接使用类名调用静态成员变；
  2. 所有对象的静态变量都是同一个；

静态函数的语法特点

在函数的定义前加上 static 关键字，即为静态函数。

语法特点：

  1. 可以直接用类名调用，无需生成对象；
  2. 在静态函数中不能直接使用非静态的成员变量；（只有静态成员变量能在没有对象的情况下调用）
