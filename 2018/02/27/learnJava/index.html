环境变量

环境变量 通常是指在操作系统当中，用来指定操作系统运行时需要的一些参数；通常是一系列的 键值对。

Path环境变量 是操作系统外部命令搜索路径。（在cmd中输入的命令其实就是运行的exe文件名称，而系统就是在Path的路径中按照名称搜索该文件然后执行）

classpath环境变量 是类文件搜索路径。（同命令运行方式，类文件运行也需要在环境变量的路径中搜索 class 文件；另相对路径中“.”代表当前目录）

JDK

bin 目录下包含 Java 所需的全部命令文件。

demo 和 sample 目录下是一些代码实例。

include 目录下是一些c语言文件。

jre 目录下是 Java 运行时环境

lib 目录下是 Java 所需的包文件

src.zip 中是 Java JDK 的一部分源文件

JRE

JRE 是 Java Runtime Environment 即 Java 运行时环境 ，包括以下几个部分：

  1. Java 虚拟机；
  2. Java 平台核心类文件；
  3. 其他支持文件；

JVM (Java 虚拟机) 是 Java Virtual Machine 的缩写，是一个由软件虚拟出来的计算机。

编译器编译的 class 文件可以在任一虚拟机上运行，而不同的系统可以有不同的虚拟机，这样就保证了 Java 程序的跨平台通用性。

变量

变量 是用来在计算机当中存储信息的工具，在 Java 语言中你需要声明语句来指定变量的存储位置和所需的内存空间。

变量的声明方法

int age; //这就是一个声明语句，“int” 指定变量的类型，“age” 指定变量的名称，“;” 表示语句结束。

该语句使计算机分配足够的空间，用于存储一个整数，而这个整数的名字叫做 age。

声明语句要解决两个问题：

  1. 变量放在哪；
  2. 变量占多大空间；

具体到该语句， int 数据类型是 Java 定义的一种数据类型，它有一个固定的范围来存储整数，由此确定了变量 age 所占空间大小。而变量名 “age” 则作为变量位置的标识，从而解决变量在哪里的问题。

变量的类型

变量的赋值

int age = 20; //"="为赋值号，“20”为变量值。

变量命名规范

  1. 变量命名规范：
    • 应该以字母，下划线或者美元符号开头；
    • 后面跟字母，下划线或者美元符号或者数字；
    • Java 变量名没有长度限制；
    • Java 变量名对大小写敏感；
  2. 驼峰命名法。

基本数据类型

布尔型变量

boolean 适用于逻辑运算，一般用于程序流程控制。

字符型变量

  • char 数据类型用来表示通常意义上的字符；
  • 字符 在这里是指用单引号包括起来的单个字符；
  • Java 字符使用 Unicode 字符集；

数值型变量

  1. 数值类型：
    • byte 占用1字节空间，范围是 -128~127；
    • short 占用2字节空间，范围是 -2^15~2^15-1;
    • int 占用4字节空间，范围是 -2^31~2^31-1;
    • long 占用8字节空间，范围是 -2^63~2^63-1;
  2. 浮点类型：
    • float 占用4字节空间，范围是 -3.403E38~3.403E8；
    • double 占用4字节空间，范围是 -1.798E308~1.798E308；

整数类型

  • Java 语言整型常量三种表现方式
    • 十进制 同正常书写；
    • 八进制 要求以0开头；
    • 十六进制 要求以0x或0X开头；
  • Java 语言的整型常量默认为 int 类型
  • 声明 long 类型可以后加“l”或“L”，如：long l = 3L

浮点类型

字面量 指数值类型在代码中的字面数量，需要注意：

  1. 整数字面量为整型（int）
  2. 小数字面量为双精度浮点型（double）

因此在给 float 类型赋值时不能直接使用字面量，而需要在其后加“F”或“f”，让字面量被识别为 float类型。

不能将范围更大的类型赋值给范围更小的类型，避免损失精度。（在进行字面量赋值时，byte 和 short 例外，但字面量大小不应超过其范围）

数值类型进行计算时，计算结果的类型以范围最大的变量类型为准。

强制转换类型可以解决赋值范围的错误，但是会造成误差，请谨慎使用。
