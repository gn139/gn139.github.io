路径相关

发表于 2018-02-27 | 分类于 知识集合

今日之拾遗，对路径相关知识的一个小总结。

  • 在 windows 系统中，\ 被用作路径分层符号，在 linux 中则是 /; 维基百科
  • ~/ 在 linux 中代表用户顶层目录；linux 中 ~/. 是什么意思
  • / 表示根目录；
  • . 当前的目录；
  • .. 上一级目录；
  • // 一般只用于使用url指定路径时使用。比如：http://......；
  • 直接以目录名开始，表示当前目录，比如：./Test 等同于 Test;
  • 相对路径 就是指由这个文件所在的路径引起的跟其它文件（或文件夹）的路径关系，以上符号常在相对路径中使用；
  • 绝对路径 则是指文件的完整路径；百度百科
