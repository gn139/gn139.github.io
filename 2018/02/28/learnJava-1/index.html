Java 中的运算符

  • 算术运算符：+, -, *, /, %, ++，--;
  • 关系运算符：>, <, >=, <=, ==, !=;
  • 布尔逻辑运算符：!, &, |, ^, &&, ||;
  • 位运算符：&, |, ^, ~, >>, <<, >>>;
  • 赋值运算符：=;
  • 扩展赋值运算符：+=, -=, *=, /=;
  • 字符串连接运算符：+;

算术运算符

  • 要点1 int i = 3 / 2; 求i值；（思考 double d = 3 / 2）

  • 要点2 i++ 与 ++i 的区别；（i++先参加运算再增大，++i先增大再参加运算）

运行以下代码可说明问题：

int i = 2;
int a = 2;
int j = i++ + 5; //可按顺序分解为 int j = i + 5; i = i + 1;
int x = ++a + 5; //可按顺序分解为 a = a + 1; int x = a + 5；
System.out.println("a = 2, i = 2, j = i++ + 5 ="+j);
System.out.println("x = ++a + 5 ="+x);
System.out.println("a = "+a);
System.out.println("i = "+i);

关系运算符

boolean b = i > j 关系运算符的运算结果为布尔值。（思考 boolean b = i == j）

逻辑运算符

!–逻辑非 &–逻辑与 |–逻辑或
^–逻辑异或 &&–短路与 ||–短路或

逻辑异或指两个关系运算同为真或同为假时其值为假，不同时为真。
使用短路与时，当第一个条件判断为假后，就停止运行，而短路或正好相反，当第一个判断为真时停止。

扩展赋值运算符

i += 5; //同 i = i + 5
i -= 5; //同 i = i - 5
i *= 5; //同 i = i * 5
i /= 5; //同 i = i / 5

表达式的类型和值

  • 表达式 是符合一定语法规则的运算符和操作符的序列。
  • 表达式的值 是对表达式中的操作数进行运算得到的结果。
  • 表达式的类型 是表达式的值的类型。

  1. 顺序结构
  2. 分支结构（即图中选择结构）
  3. 循环结构

if else 结构

语法结构（一）:

if (布尔类型表达式){
    语句一；
    语句二；
    ......
}

语法结构（二）:

if(布尔类型表达式){
    语句一；
    语句二；
    ......
}else{
    语句三；
    语句四；
    ......
}

语法结构（三）：

if(布尔类型表达式){
    语句一；
    语句二；
    ......
}else if(布尔类型表达式){
    语句三；
    语句四；
    ......
}else if(布尔类型表达式){
    语句五；
    语句六；
    ......
}else{
    语句七；
    语句八；
    ......
}

switch 结构

switch(表达式){
    case 常量1：
        语句1;
        break;
    case 常量2：
        语句2;
        break;
    ......
    case 常量N:
        语句N;
        break;
    [default:
        默认语句;
        break;]
}

其中 表达式 只允许四种类型：byte，int，char，short（在 Java 的1.7之后的 Jdk 版本，Java 中的 switch 里面表达式的类型可以是 string 类型。）

注意：如果没有 break 一旦表达式的值匹配正确后，假设值为常量2就会运行语句2~语句N，不再进行判断直到下一个break或没有语句后为止。

循环结构

此处为图

for 循环语句

for (赋值;布尔表达式;变化){
    语句1;
    ......
    语句N;
}

大括号内为 循环体。执行过程为，先赋值再执行布尔表达式，当为真时执行循环体并执行变化语句，当为假时跳过循环体并停止循环。

while 循环语句

while(布尔表达式){
    语句1;
    ......
    语句N;
}

执行过程为，先执行布尔表达式，当为真时执行循环体再执行布尔表达式，直到表达式的值为假时停止循环。
